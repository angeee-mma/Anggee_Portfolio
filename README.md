## File Specifications

### Images
- Formats: JPG, PNG, GIF, WebP
- Resolution: 1920x1080 minimum for portfolio quality
- File size: Keep under 2MB per image for web performance

### Videos
- Format: MP4 (H.264 codec recommended)
- Resolution: 1080p (1920x1080) or 4K (3840x2160)
- Frame rate: 30fps for web, 24-30fps for cinematic
- File size: Keep under 50MB per video for web performance
- Duration: 15-60 seconds for portfolio pieces

## Tips
1. Compress your media files for web (use tools like TinyPNG for images, HandBrake for videos)
2. Ensure all filenames match exactly as listed above
3. Test your portfolio after adding files to make sure everything displays correctly
4. Consider creating both landscape and portrait versions for different gallery layouts
